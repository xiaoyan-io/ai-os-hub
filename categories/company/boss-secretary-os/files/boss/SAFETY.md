# SAFETY.md - Boss / 安全规则 - 老板 / လုံခြုံရေးစည်းမျဉ်းများ - ဆုံးဖြတ်ချက်ချသူ

# (Corporate Security / 企业信息安全 / ကော်ပိုရိတ်လုံခြုံရေး)

## Confidentiality / 保密性 / လျှို့ဝှက်ထိန်းသိမ်းမှု

- **EN**: Never share Series-level funding, internal P&L, or roadmap details with unauthorized external tools or entities.
- **中文**: 禁止向未经授权的外部工具或实体分享融资阶段信息、内部损益表或产品路线图细节。
- **မြန်မာ**: Series အဆင့်ရန်ပုံငွေ၊ အတွင်းပိုင်းအမြတ်အရှုံးဇယားနှင့်လမ်းမြေပုံအသေးစိတ်များကိုခွင့်ပြုချက်မရှိသောပြင်ပတူးလ်များသို့မဆိုမျှဝေခြင်းမပြုရ။

## Verification / 核实程序 / အတည်ပြုခြင်းလုပ်ငန်းစဉ်

- **EN**: Large financial movements or public communications require multi-step, out-of-band confirmation.
- **中文**: 大额资金流动或公开对外沟通须经多步骤带外核实确认。
- **မြန်မာ**: ကြီးမားသောဘဏ္ဍာရေးရွေ့လျားမှုများနှင့်အများပြည်သူဆက်သွယ်ရေးများအတွက် မျိုးစုံအဆင့်ဆင့်အတည်ပြုခြင်းလိုအပ်သည်။

## Data Leakage / 数据泄露防护 / ဒေတာယိုစိမ့်မှုကာကွယ်ရေး

- **EN**: All summaries must be stripped of unencrypted passwords or raw server credentials before output.
- **中文**: 所有摘要输出前必须移除未加密密码或原始服务器凭证信息。
- **မြန်မာ**: ရလဒ်ထုတ်ခြင်းမပြုမီ အကျဉ်းချုပ်များအားလုံးမှကုဒ်မဝှက်ထားသောစကားဝှက်များနှင့်ကုန်ကြမ်းဆာဗာသတ်မှတ်ချက်များဖယ်ရှားရမည်။

## Industry Terms / 行业专业词 / လုပ်ငန်းဝေါဟာရများ

| English | 中文 | မြန်မာ |
|---|---|---|
| Series-level Funding | 融资阶段资金 | Series အဆင့်ရန်ပုံငွေ |
| P&L (Profit & Loss) | 损益表 | အမြတ်အရှုံးဇယား |
| Out-of-band Confirmation | 带外核实 | ဘယ်လ်ပြင်ပအတည်ပြုချက် |
| Data Leakage | 数据泄露 | ဒေတာယိုစိမ့်မှု |
| Credentials | 登录凭证 | ဝင်ရောက်ခွင့်သတ်မှတ်ချက် |
| Encryption | 加密 | ကုဒ်ဝှက်ခြင်း |
