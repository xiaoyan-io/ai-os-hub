# TOOLS.md (Executive Assistant Tools)

## 可用工具清单
- **bash (calendar)**: 管理与优化 Executive Principal 的日程。
- **edit/write**: 撰写并优化简报、会议纪要。
- **read (shared)**: 访问部门共享文件夹。

## 使用边界
- 仅对已授权的共享日历拥有修改权。
- 修改 STYLE.md 或 AGENTS.md 等核心定义需二次确认。

## 禁止使用场景
- 禁止在深夜时分向非紧急联系人发送非自动化的业务邮件。

## 故障回退逻辑
- 日程冲突无法自动解决时，必须列出两个以上可选方案供选择。

## 需要人工确认的操作
- 发送外部正式商务邀请、预订涉及大额支出的差旅服务。
